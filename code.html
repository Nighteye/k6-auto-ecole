Sur une période de 5 jours (de 9h à 12h et de 13h à 17h) soit pendant 35 heures, K6 auto- école vous propose la formation au Code de la route en accéléré avec un enseignant vous accompagnant tout au long du stage.

Thème par thème, vous évoluerez progressivement pour maitriser le code de la route.

Déroulement

La formation se déroule en alternant les cours et les tests afin de mieux vous perfectionner. Vous passerez ensuite un examen de code blanc qui confirmera vos acquis avant de passer l'examen final dans les jours qui suivent. Si toutefois l'examen blanc n'a pas été favorable, vous pourrez continuer à compléter votre formation gratuitement en salle de code.

Cette formule comprend :

  • La Formation théorique
  • 1 livret Code
  • 1 Présentation à l'examen du Code de la Route
  • 1 Accompagnement à l'examen de Conduite
  • 1 Pack Web accès illimité au Code
